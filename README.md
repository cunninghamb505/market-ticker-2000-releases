# Market Ticker 2000

A scrolling stock ticker bar for Windows that sits on top of all your windows.

**Author:** Brandon Cunningham

![Windows](https://img.shields.io/badge/platform-Windows-blue)

## Features

- Real-time stock quotes (SPY, DOW, NQ, AAPL, MSFT, BTC, ETH and more)
- Sparkline mini-charts next to each symbol
- Market status indicator (Open / Closed / Pre-Market / After-Hours)
- 4 color themes: Dark, Light, Bloomberg, Ocean
- Click any symbol to open its Yahoo Finance chart
- Pause scrolling on hover
- Flash highlight when prices change
- Breaking news headlines from Yahoo Finance / MarketWatch
- Price alerts with system tray notifications
- Global hotkey (Ctrl+Shift+T) to show/hide
- Auto-hide after configurable delay
- Multi-monitor support
- Draggable bar positioning
- Full settings dialog (General, Appearance, Features, Alerts)

## Usage

1. Run `MarketTicker2000.exe`
2. A ticker bar appears at the top of your screen
3. Right-click the system tray icon (near the clock) for the menu
4. Use **Settings** to customize symbols, colors, themes, and features
